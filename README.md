# DreamDiary
夢日記をつけるアプリ

# 開発環境
XCode 11.3
Swift 5.0
RealmSwift 4.4.1
Firebase 

# 導入手順

以下の手順に従ってローカル環境でビルドを行ってください。

クローンしてくる
```
$git clone https://github.com/maip0902/DreamDiary.git
```

ディレクトリ移動
```
$cd DreamDiary
```

pod init

```
$pod init
```

```
$pod install
```

＊必要に応じて
```
pod update
```
